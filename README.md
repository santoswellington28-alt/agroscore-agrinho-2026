# AgroScore - Pegada Ecológica

Projeto interativo desenvolvido para o Concurso Agrinho Paraná 2026.

## Tema
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo
Criar um site interativo que avalia a pegada ecológica e o nível de sustentabilidade de propriedades agrícolas ligadas às culturas de cana-de-açúcar, soja e milho.

## Problema trabalhado
Uso eficiente da água na irrigação, manejo e conservação do solo, reaproveitamento de resíduos da colheita (biomassa) e transição para fontes de energia limpa e renovável no campo.

## Tecnologias utilizadas
- HTML5 (Estruturação do conteúdo)
- CSS3 (Estilização e design responsivo focado no tema ambiental)
- JavaScript (Lógica do quiz e cálculo automático da pontuação ecológica)
