# AgroScore - Pegada Ecológica

Projeto desenvolvido para o Concurso Agrinho Paraná 2026.

## Tema
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo
Criar um site interativo que avalia a pegada ecológica de empresas ligadas à cana-de-açúcar, soja e milho.

## Problema trabalhado
Uso de água, solo, energia, resíduos, transporte e preservação ambiental no agro.

## Tecnologias
HTML, CSS e JavaScript.
